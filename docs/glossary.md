# 术语表

平台与文档中反复出现的专业术语，统一解释。

## 间隔重复相关

- **SRS（Spaced Repetition System，间隔重复系统）**
  核心思想是"在你快要忘记的临界点复习"，以最小复习成本牢固记忆。Anki 的底层机制。

- **FSRS（Free Spaced Repetition Scheduler）**
  现代间隔重复排期器，用记忆曲线模型预测每张卡"何时会被遗忘"，据此安排复习。
  比传统算法更科学、复习量更省。本平台默认算法，用 `ts-fsrs` 库实现。

- **SM-2**
  Anki 早期采用的经典间隔重复算法，靠 ease factor（难度系数）和 interval（间隔）递推。
  简单、社区熟悉，作为可选算法。

- **ease factor（难度系数）**
  SM-2 中表示某张卡"有多容易记"的系数，答得越好系数越高、间隔增长越快。**平台默认对用户隐藏。**

- **interval（间隔）**
  下次复习距今的天数。算法根据作答质量动态调整。

- **retention（留存率）**
  复习时答对的比例，反映学习质量。统计模块核心指标之一。

- **卡片成熟度（maturity）**
  卡片按间隔长短分为新卡 / 学习中 / 已成熟。间隔越长越"成熟"，代表记得越牢。

## 卡片与牌组

- **牌组（deck）**
  一组卡片的集合，通常按主题/词库划分（如"考研核心词"）。

- **卡片（card）**
  最小复习单位。常见类型：正反词卡（正面词/背面释义）、cloze、释义卡。

- **cloze（填空卡）**
  挖空填空型卡片，把句子中某些词遮住让你回忆，适合在语境中记忆。

- **`.apkg`**
  Anki 的牌组导出格式，本质是一个 zip 包，内含一个 SQLite 数据库 + 媒体文件。

## 语言学习方法

- **语块（chunk / chunking）**
  大脑以高频搭配整块存储语言（如 `the more... the more...`）。背熟整段即把语块连语境一起记住。

- **可理解输入（comprehensible input，i+1）**
  Krashen 的假说：输入难度略高于当前水平（i+1）时习得效率最高。对应"生词可控"。

- **自动化（automaticity）**
  把语言从"需要思考的知识"练成"不假思索的本能"，即流利度的本质。

- **影子跟读（shadowing）**
  边听边几乎同步地模仿跟读，强化语音语调与流利度。文章学习器 TTS 跟读对应此法。

- **sentence mining（句式挖掘）**
  从真实语料中挑出有价值的句式/语块，针对性地记忆。

## 平台内部概念

- **语言包（Language Pack）**
  某门学习语言的全部"语言相关"配置：词频分级表、语法点清单、TTS 语音、生成 prompt 模板。
  加一门语言 = 加一份语言包。

- **学习语言 vs 界面语言**
  **学习语言**：用户正在学的外语（如英语、日语），存于用户学习档案。
  **界面语言**：平台 UI 用哪种语言显示（next-intl 的 `en`/`zh`）。两者独立。

- **计时器适配器（Timer Adapter）**
  统一的计时抽象，被刷卡 / 阅读 / 测验等场景接入，各场景独立计时并可汇总为总学习时长。

- **记忆锚点（memory anchor）**
  AI 为难词生成并固化到卡片的助记 / 辨析内容，复习时复现，帮助牢记。
